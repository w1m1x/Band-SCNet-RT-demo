# Demo deployment notes

This folder is ready to publish as a static website.

## Recommended: GitHub Pages

1. Create a new public GitHub repository, for example `paper-demo`.
2. Upload all files in this folder to the root of that repository:
   - `index.html`
   - `audio/`
   - `manifest.csv`
   - `.nojekyll`
   - `README.md`
3. Open the repository `Settings` page.
4. Open `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/root`, then save.
7. After GitHub finishes publishing, the public URL will usually be:
   `https://YOUR_GITHUB_USERNAME.github.io/paper-demo/`

## Fast alternative: Netlify Drop

1. Go to https://app.netlify.com/drop
2. Drag this whole folder into the page.
3. Netlify will give you a public URL immediately.
